# Setting Print akta pada lembar A3

  1. Klik menu file dan pilih print atau Ctrl+P.
  2. Pada Tab General, Pilih printer Officejet 7610 series pada kolom Printer.
  3. Pilih menu Properties....
  4. Klik Tab Paper/Quality, ubah settingan seperti gambar dibawah ini :
  5. Klik Tab Layout, ubah settingan seperti gambar dibawah ini :
